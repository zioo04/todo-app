# To-Do List Project

이 프로젝트는 간단하지만 실용적인 할 일 관리 웹 애플리케이션입니다.

### 💡 주요 특징 및 기술적 시도
- **효율적인 도구 활용:** ChatGPT를 활용해 핵심 로직을 신속하게 구현하고 배포 프로세스를 익혔습니다.
- **사용자 경험(UX) 개선:** 기본 아이콘을 사용자 친화적인 ✅ 아이콘으로 커스텀하여 직관적인 UI를 구성했습니다.
- **웹 호스팅:** GitHub Pages를 통해 실시간 웹 서비스를 배포하는 전 과정을 경험했습니다.

### 🛠 기술 스택
- HTML5, CSS3, JavaScript
- GitHub Pages
